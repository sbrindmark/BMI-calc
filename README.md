# BMI Kalkylator

En enkel konsolapplikation för att beräkna Body Mass Index (BMI) med både metriska och imperiala enheter.

## Funktioner

- Tar emot vikt och längd från användaren.
- Stöd för både metriska (kg, meter) och imperiala (lbs, inches) enheter.
- Felhantering för ogiltig inmatning.
- Visar BMI enligt olika beräkningssätt.
